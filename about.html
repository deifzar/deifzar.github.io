Hi, I'm David - a security consultant evolving into security architecture and DevSecOps engineering

With over 15 years of hands-on penetration testing and offensive security experience, I'm transitioning into security architecture and DevSecOps. Currently as a Principal Security Consultant, I'm channeling my offensive security expertise into building automation platforms that embed security testing into the development lifecycle. Through personal projects like CPTM8—a continuous penetration testing platform built with Go microservices and Kubernetes—I'm demonstrating how years of breaking systems translates into building more secure ones.

My expertise spans offensive security (OSCP, Red Team Operator), cloud security (AWS & Azure certified), and container orchestration. I've architected a continuous penetration testing platform using Go microservices, Kubernetes, and modern DevOps practices. From vulnerability scanning automation to security tool integration, I'm applying this knowledge to help teams shift security left without sacrificing velocity.
